Events

Key Event

This event logs information about a specific key pressed from the keyboard, including the following attributes:

  • Code of key pressed
  • Timings
    • Timestamp of keydown
    • Timestamp of keyup
    • Duration of keypress
  • Keypress meta
    • Was Alt held down during the keypress?
    • Was Ctrl held down during the keypress?
    • Was Shift held down during the keypress?
    • Was ⊞ or ⌘ Command held down during the keypress?
    • How many characters were typed by holding the key down?

Word Event

This event logs aggregate information about typing a word. This includes the following attributes:

  • Sequence of keys entered
  • Final entered string
  • Number of mistakes
  • Event timings
    • Timestamp of word start
    • Timestamp of word end
    • Duration of typing the word
    • Downtime per word (time spent not pressing any keys, may be negative)
  • Key timings
    • Total keypress duration
    • Average keypress duration
    • Number of keypresses
